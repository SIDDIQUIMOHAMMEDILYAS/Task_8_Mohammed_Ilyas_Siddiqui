## ğŸ“¦ Installation & Usage

Since this is a standalone HTML application, no installation is required.

1.  **Download** the `index.html` file.
2.  **Open** the file in any modern web browser (Chrome, Edge, Firefox, Safari).
3.  **Start** adding your tasks!

### How to Use

1.  **Add a Task:** Click the **"+"** floating button or press `Enter` in the search bar (custom logic). Fill in the title, select category, priority, and due date.
2.  **Manage Tasks:** Check the box to mark as completed. Hover over a task to see the **Edit** or **Delete** buttons.
3.  **Filter:** Use the left sidebar to filter by category.
4.  **Sort:** Click and drag the handle on the left of any task to reorder your list.
